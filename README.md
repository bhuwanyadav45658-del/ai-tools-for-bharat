# AI Tools for Bharat

Frontend-first SEO utility website for Indian AI prompt generators, templates, mini-tools, and workflow pages.

## Run locally

```bash
npm start
```

Open:

```text
http://localhost:4173
```

## What is included

- Homepage with search, popular tools, profession browsing, languages, task chips, trust badges, SEO content, and FAQ.
- Clean route-style static pages for about, contact, privacy, terms, disclaimer, advertise, submit tool, sitemap, all tools, categories, prompts, templates, guides, languages, favorites, and recent tools.
- 10 category routes for students, teachers, small business, job seekers, office employees, accountants, HR, creators, freelancers, and sales/marketing.
- First 15 tool pages with interactive forms, generated prompts/templates, copy, markdown copy, TXT/MD/JSON download, print, share buttons, AI open buttons, feedback, favorites, recent tools, and safety notices.
- Regional language hubs for Hindi, Telugu, Tamil, Kannada, Marathi, Bengali, Malayalam, Gujarati, and Hinglish.
- Prompt collection pages, template download pages, generic guide pages, and comparison pages.
- Static data files in `src/data`.
- `robots.txt`, `sitemap.xml`, and a small clean-URL Node server.

## Notes before publishing

- Replace `https://example.com` in `robots.txt` and `sitemap.xml` with the real domain.
- Replace the placeholder email on the Contact page.
- Add analytics or ads only after privacy text is updated.
- Keep tool pages useful: examples, FAQs, safety notes, related links, and real mini-tool behavior.
